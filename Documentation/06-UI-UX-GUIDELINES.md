# UI/UX Design System: LinuxGit

## Design Philosophy: "Flow State"

Our design philosophy centers around enabling developer "flow state" - an immersive, distraction-free experience where the interface becomes invisible and developers focus purely on their work.

### Core Principles

1. **Liquid Depth** - Interfaces that feel alive with subtle depth, light, and motion
2. **Flow Visualization** - Git history as flowing rivers, not static diagrams
3. **Ambient Intelligence** - AI assistance that feels natural, not intrusive
4. **Chromatic Expression** - Rich, vibrant colors that convey meaning and emotion
5. **Keyboard Velocity** - Every action achievable without touching the mouse

## Liquid Glass Design System

Inspired by Apple's iOS 26 "Liquid Glass" and Microsoft's Fluent Acrylic, our components feel fluid, dimensional, and alive.

### Glass Card Component

```css
.glass-card {
  /* Multi-layer glass effect */
  background: linear-gradient(
    135deg,
    rgba(255, 255, 255, 0.08) 0%,
    rgba(255, 255, 255, 0.04) 50%,
    rgba(255, 255, 255, 0.02) 100%
  );

  /* Frosted glass blur */
  backdrop-filter: blur(20px) saturate(180%);
  -webkit-backdrop-filter: blur(20px) saturate(180%);

  /* Subtle border for definition */
  border: 1px solid rgba(255, 255, 255, 0.08);

  /* Soft inner glow */
  box-shadow:
    inset 0 1px 0 rgba(255, 255, 255, 0.1),
    0 8px 32px rgba(0, 0, 0, 0.3),
    0 2px 8px rgba(0, 0, 0, 0.2);

  /* Smooth corners */
  border-radius: 16px;

  /* Dynamic light reflection (on hover) */
  &:hover {
    background: linear-gradient(
      135deg,
      rgba(255, 255, 255, 0.12) 0%,
      rgba(255, 255, 255, 0.06) 50%,
      rgba(255, 255, 255, 0.03) 100%
    );

      inset 0 1px 0 rgba(255, 255, 255, 0.15),
      0 12px 40px rgba(0, 0, 0, 0.35),
      0 4px 12px rgba(0, 0, 0, 0.25);
  }
}
```

## Git Flow Visualization: "River System"

### The Problem with Traditional Git Graphs

Traditional Git visualization uses circles and lines - a paradigm from the 1990s. This creates:
- Visual clutter with many branches
- No sense of "flow" or direction
- Difficulty tracking merge points
- Poor emotional connection to code history

### Our Solution: River Flow Visualization

Commits flow like water through a river delta system. Branches are streams that diverge and merge, with width indicating activity intensity.

## Accessibility

### Focus States

```css
/* Visible, beautiful focus rings */
:focus-visible {
  outline: none;
  box-shadow:
    0 0 0 2px var(--bg-deep),
    0 0 0 4px var(--accent-primary),
    0 0 12px var(--accent-primary-glow);
}
```

### Color Contrast

All text combinations meet WCAG 2.1 AA standards:
- Primary text on background: 15:1 ratio
- Secondary text on background: 7:1 ratio
- Accent colors tested for deuteranopia/protanopia

### Reduced Motion

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```
